# RoboArm Controller

Android app + ESP32 Web UI controller for an ESP32-based robotic arm.

## Features
- Control ESP32 robotic arm over WiFi
- Android app with built-in WebView
- Clean UI
- Shows warning when ESP32 is not connected
- Works without internet (local WiFi)

## Credits
- Original ESP32 Robot Arm project: Tech Talkies

## Contents
- ESP32 Arduino code
- Android app source code
- Web UI files

## How to use
1. Flash the ESP32 with the provided Arduino code
2. Power on the ESP32 and connect your phone to its WiFi
3. Open the RoboArm Controller app
4. Control the robotic arm from your phone
